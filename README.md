# Glider Range Optimization

MATLAB-based project for designing and analyzing an unpowered glider to achieve the maximum flight range under predefined design constraints.

## Project Description

This project investigates the conceptual design and performance of a high-altitude glider using numerical simulation and flight mechanics principles.

### Design Constraints

- No thrust
- Operating altitude: 15 km
- Mass: 3 kg
- Wing span: 10 m
- Aspect ratio (AR) ≤ 10
- Same airfoil along the wing
- Wing incidence angle: 0°
- Tip chord constraint: Ct ≥ (1/3)Cr

## Features

- Aircraft geometry definition
- Atmospheric model
- Aerodynamic calculations
- Flight trajectory simulation
- Range optimization
- Performance visualization

## Tools

- MATLAB
- Flight Mechanics
- Aerodynamics
- Numerical Methods
